Html Cheatsheet
  1. HTML BOILER PLATE:

    This is Basic HTML Boiler Plate

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Title Of The Page</title>
</head>
<body>
    
</body>
</html>
  2. META KEYWORDS TAG:

    Keywords are special words that allows user search for a specific website.

    <meta name= "keywords" content= "html cheatsheet, best html cheatsheet">
  3. META DESCRIPTION TAG:

    To Add a Description of our Website or a Page, We Use this Tag.

    <meta name= "description" content = "This is a description">
  4. META INDEX TAG:

    If You Want Your Website to be Indexed in Google and the Bots Should Follow all the Links Present Use this Tag.

    <meta name= "robots" content= "INDEX, FOLLOW">
  5. CSS LINK:

    These Tag is Used To Link an External Style Sheet.

    <link rel="stylesheet" href="style.css">
  6. JS LINK:

    These Tag is Used To Link an External JavaScript.

    <script src="index.js"></script>
  7. HEADINGS TAG:

    This is primarily a Heading. There are 6 Types Of Heading:

    <h1>

    Keshav

    Keshav

<h5>
    Keshav
    </h5>

    Keshav
    </h6>   <!--Smallest-->
  8. PARAGRAPH TAG:

    This is a Paragraph. Lorem 40 would give us a Paragraph of 40 words

    <p>lorem40</p>
  9. STRONG TAGS:

    Anything Written in Strong tags would make the text Bold

    <strong>Keshav</strong>
  10. EM TAGS:

    Anything Written in EM tags would make the text Italics

    <em>Keshav</em>
  11. BR TAGS:

    BR Tags(<br>)Create a Division in the line

    Keshav<br>Ladha
Would create a Sentence Like:
Keshav
    Ladha
  12. ANCHOR TAGS:

    ANCHOR Tags(<a href=""></a>) are Links To Other Websites or Pages in Your Website

    1. <a href="www.google.com">GO TO GOOGLE</a>
By clicking on GO TO GOOGLE the anchor tag would navigate you to google.com
      2. 


    2. <a href="index.html" target="_blank">GO TO INDEX.HTML</a>
This would take you to INDEX.HTML and "target=_blank" would make the website open in a separate window
  13. IMG TAGS:

    IMG Tags(<img src="" alt="">) are Links To Images Either Present On The Web Or in Your Folder.

    src = "takes the source of the image either local or on web"
alt = "takes the Sentence to be shown to the user when the image has not loaded properly"

<img src="Keshav.png" alt="Keshav Logo">
This is example of an Image present in the Folder

<img src="www.link-to-the-image.com" alt="Random Image">
This is an example of an Image taken from the web
  14. OL TAG:

    This is an Ordered List in Which we can Store List Items with an <li> tag 

    <ol type='1', 'A', 'a', 'I', 'i'>

        <li>LIST ITEM 1</li>
    <li>LIST ITEM 2</li>
    <li>LIST ITEM 3</li>
    
</ol>
  15. UL TAG:

    This is an Unordered List in Which we can Store List Items with an <li> tag 

    <ul>

        <li>LIST ITEM 1</li>
    <li>LIST ITEM 2</li>
    <li>LIST ITEM 3</li>
    
</ul>
  16. TABLES:

    This is a Table

    <table>
    <thead>    <!--'thead' consists of table headings-->              
        <tr>
            <th>Name</th>       <!--TABLE HEADING (<th></th>) TAKES THE HEADING OF THE TABLE. TEXT IN THE HEADING APPEARS BOLDER-->
            <th>Employee Id</th>
            <th>Employee Role</th>
        </tr>
    </thead>
    <tbody>    <!--'tbody' consists of table body-->    
        <tr>
            <td>Keshav</td>   <!--TABLE DATA (<td></td>) ROW TAKES THE DATA OF THE TABLE-->   
            <td>3483 </td>
            <td>HTML Expert </td>
        </tr>
        <tr>
            <td>Devesh </td>
            <td>7564 </td>
            <td>Accounts Manager</td>
        </tr>
    </tbody>
</table>
