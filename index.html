Html Cheatsheet
  1. HTML BOILER PLATE:

    This is Basic HTML Boiler Plate

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Title Of The Page</title>
</head>
<body>
    
</body>
</html>
  2. META KEYWORDS TAG:

    Keywords are special words that allows user search for a specific website.

    <meta name= "keywords" content= "html cheatsheet, best html cheatsheet">
  3. META DESCRIPTION TAG:

    To Add a Description of our Website or a Page, We Use this Tag.

    <meta name= "description" content = "This is a description">
  4. META INDEX TAG:

    If You Want Your Website to be Indexed in Google and the Bots Should Follow all the Links Present Use this Tag.

    <meta name= "robots" content= "INDEX, FOLLOW">
  5. CSS LINK:

    These Tag is Used To Link an External Style Sheet.

    <link rel="stylesheet" href="style.css">
  6. JS LINK:

    These Tag is Used To Link an External JavaScript.

    <script src="index.js"></script>
  7. HEADINGS TAG:

    This is primarily a Heading. There are 6 Types Of Heading:

    <h1>
    Keshav
    </h1>   <!--Largest-->
<h2>
    Keshav
    </h2>
<h3>Keshav</h3>
<h4>Keshav</h4>
<h5>Keshav</h5>
<h6>Keshav</h6>   <!--Smallest-->
  8. PARAGRAPH TAG:

    This is a Paragraph.

    <p>Keshav</p>
  9. OL TAG:

    This is an Ordered List in Which we can Store List Items with an <li> tag 

    <ol type='1', 'A', 'a', 'I', 'i'>
  10. 

        <li>LIST ITEM 1</li>
    <li>LIST ITEM 2</li>
    <li>LIST ITEM 3</li>
    

    11. </ol>
  11. UL TAG:

    This is an Unordered List in Which we can Store List Items with an <li> tag 

    <ul>
  12. 

        <li>LIST ITEM 1</li>
    <li>LIST ITEM 2</li>
    <li>LIST ITEM 3</li>
    

    13. </ul>
